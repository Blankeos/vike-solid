# Contributing

## Building

Build `vike-solid`:

```bash
git clone git@github.com:magne4000/vike-solid
cd vike-solid/ && pnpm install
pnpm build
cd ../
```

## Validating

### Running the examples

You can then test your modifications against an example, e.g. `examples/basic/`:

```bash
cd examples/basic/
pnpm dev
cd ../../
```
